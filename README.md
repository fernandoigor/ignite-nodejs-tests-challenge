# Ignite - Rocketseat
### Desafio 01 - Teste Unitários

#### Sobre o desafio

Nesse desafio, você deverá criar testes unitários para uma aplicação já pronta.

#### Teste
 - **Yarn**
 	- Install
``
yarn
``
 	- Teste Jest
``
yarn test
``
 - **NPM**
 	- Install
``
npm install
``
 	- Teste Jest
``
npm run test
``
